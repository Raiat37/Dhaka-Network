# Dhaka Network Simulation

## 📌 Overview
This project is a network simulation built using Cisco Packet Tracer. It models a structured network design for Dhaka, focusing on connectivity, routing, and efficient communication between multiple nodes.

The simulation demonstrates how a real-world network infrastructure can be designed and tested in a virtual environment.

---

## 🗺️ Network Topology
The network consists of interconnected devices such as routers, switches, and end devices organized to simulate a scalable and efficient architecture.

### Key Features:
- Hierarchical network design
- Multiple subnetworks
- End-to-end connectivity
- Scalable structure

---

## 🖥️ Devices Used
- Routers  
- Switches  
- PCs / End Devices  
- Servers (if applicable)

---

## ⚙️ Configuration Summary
The following configurations are implemented in the network:

- IP Addressing & Subnetting  
- Routing (Static / Dynamic depending on your setup)  
- VLANs (if used)  
- Basic network security configurations  
- Device-to-device communication setup
